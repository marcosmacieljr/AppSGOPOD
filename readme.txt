download instaladores
----------------------------------
node 10x: https://nodejs.org/dist/latest-v10.x/
expo (app): 
(android) https://play.google.com/store/apps/details?id=host.exp.exponent&hl=pt_BR
(ios) https://apps.apple.com/br/app/expo-client/id982107779

comandos npm
----------------------------------
instalar expo: npm install expo-cli --global
instalar expo sqlite: expo install expo-sqlite
instalar dependências diversas: npm install
iniciar: expo start
